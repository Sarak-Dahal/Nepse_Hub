Help and Support

  • What is Nepse Hub?

    Nepse Hub is a complete package required for an investor to lookup upon Nepali Share Market. Where an investor can gain access to multiple tools and data from Nepali Stock Exchange free of cost.

  • Why Nepse Hub?

    Nepse Hub in near future will provide all necessary info regarding stock market of Nepal in your fingertips. With easy, responsive User Interface and with more accuracy. The site will be your Hub and guide your path to surf NEPSE and invest in realtime.

  • Does the site work on Mobile Device too?

    Yes of course, the website is built responsive so it work on all available platform/devices. Soon we will even be available in Google Play Store and Apple Store.

  • Is data displayed similar to NEPSE data?

    Yes with 100% confirmation, all data displayed here is realtime NEPSE data with 0 second interval time. Data displayed here will be as similar to data displayed in NEPSE official website.

  • What is Refresh Data in Live Data?

    Refresh Data is a button while on clicking the button, it takes all the live data of NEPSE with 0 second interval time.

  • What is Predict in Prediction?

    Predict is a button with an input field next to it which provides prediction of Stock you enter. Prediction are based on stocks past data and even.

  • What should I input in Prediction?

    In prediction we have an input field where you should input EBL for Everest Bank Limited.

  • What is Re-Train Model?

    Re-Train Model is a button which first display Stock Data date right above it. As stock model are trained accordingly and if you want model to be upto date and gain more accuracy you can click Re-Train Model. It trains model as of todays date and give more accurate data as a result.

  • What is Calculate in Market Tool?

    Calculate is a button which provides user to preform stock averaging with input field above.
